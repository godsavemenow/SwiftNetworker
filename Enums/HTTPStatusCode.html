HTTPStatusCode

public enum HTTPStatusCode : Int

An enumeration representing HTTP status codes.

Overview

The HTTPStatusCode enumeration defines a comprehensive list of HTTP status codes categorized into success, redirection, client error, and server error responses. Each case in this enumeration represents a specific HTTP status code and its associated integer value. The enumeration also provides utility methods to check the category of a given status code and to create an instance from an integer value.

This enumeration is useful for handling and interpreting HTTP responses in network operations, allowing developers to easily identify the type of response and take appropriate action based on the status code.

Usage

To use the HTTPStatusCode enumeration, either match against specific cases or use the utility methods to check the category of a status code. For example:

if HTTPStatusCode.isSuccessful(response.statusCode) {
    // Handle successful response
} else if HTTPStatusCode.isClientError(response.statusCode) {
    // Handle client error response
}
