NetworkErrorCases

public enum NetworkErrorCases : CustomStringConvertible

An enumeration representing various types of network-related errors, each with a custom description.

Overview

The NetworkErrorCases enumeration defines a wide range of network-related errors, each with a specific description. These error cases cover common network issues such as invalid URLs, no data responses, timeouts, client and server errors, parsing and decoding errors, and more. Each case can provide a detailed description that can include additional context or messages to aid in understanding the nature of the error.

This enumeration is designed to be used with the NetworkError struct to provide a standardized way to handle network errors across different parts of an application.

Usage

To use the NetworkErrorCases enumeration, create an instance of the NetworkError struct with a specific case and optional message. Use the description property to get a textual representation of the error case.

let errorCase = NetworkErrorCases.badRequest("Invalid parameters")
let error = NetworkError(errorCase: errorCase, apiErrorMessage: "Parameter 'id' is missing")
print(error.detailedDescription)  // Output: "Bad Request: Invalid parameters - Parameter 'id' is missing"
